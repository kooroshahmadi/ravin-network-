# Port Numbers Cheat Sheet

| Protocol       | Port     | Type |
| :------------- | :------- | :--- |
| **FTP**        | 20, 21   | TCP  |
| **SSH / SFTP** | 22       | TCP  |
| **Telnet**     | 23       | TCP  |
| **SMTP**       | 25       | TCP  |
| **DNS**        | 53       | UDP  |
| **DHCP**       | 67, 68   | UDP  |
| **TFTP**       | 69       | UDP  |
| **HTTP**       | 80       | TCP  |
| **POP3**       | 110      | TCP  |
| **NTP**        | 123      | UDP  |
| **IMAP**       | 143      | TCP  |
| **SNMP**       | 161, 162 | UDP  |
| **HTTPS**      | 443      | TCP  |
| **SMB**        | 445      | TCP  |
| **RDP**        | 3389     | TCP  |
