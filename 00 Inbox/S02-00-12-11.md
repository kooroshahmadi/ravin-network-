Securing console port:
first we go to global mode.
to config the device hostname
	Switch(config)# hostname SW-1

to set a password to enter:
	Switch(config)# line console 0
	Switch(config-line)# password cisco123
	Switch(config-line)# login (enable password checking)
	from now on if someone connects through the console port, they must enter the password.

to set a timeout for CLI:
	Switch(config-line)# exec-timeout 1 (the number is in minutes)

to set port speed:
	Switch(config-line)# speed 57600

to set a password to enter Enable Mode:
	Switch# enable password cisco123

to encrypt passwords so that no one can see it we use:
	Switch(config)# service password-encryption

"no" command in cisco CLI is equal to delete to remove:
	Switch(config)# no serviec password-encryption
	will remove password encryption

instead of a password we can user secret: this will hash the password
